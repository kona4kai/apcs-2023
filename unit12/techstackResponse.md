## Technologies

 - [Premiere Pro](https://www.adobe.com/products/premiere.html) is mostly written in C++, however plugins and APIs can be coded in C/C++, JavaScript, and ExtendScript.
[Adobe Support Community](https://community.adobe.com/t5/premiere-pro-discussions/premiere-pro-programming-language/td-p/8721554)

 - [Blender](https://www.blender.org/) has a core of C, but uses C++ for physics, audio, game-engine and Cycles render engine. It also uses Python for Scripting.
 [Quora](https://www.quora.com/What-programming-languages-are-software-like-Photoshop-Premiere-Pro-Cinema4D-and-Blender-written-in)

  - [Visual Studio Code](https://code.visualstudio.com/) VS Code is programmed in TypeScript, as it allows for distribution across platforms using Electron. Use of standard web APIs also allows VS Code to work online for GitHub CodeSpaces.
